How to format a cron schedule

A cron calendar is made up of 5 fields:

  • Minute: It can be a specific minute or every few minutes. Show usage
    • * Any value
    • , Multiple values
    • - Range of values
    • / Step values (every certain amount)
  • Hour: It can be a specific hour or every few hours. Show usage
    • * Any value
    • , Multiple values
    • - Range of values
    • / Step values (every certain amount)
  • Day (month): It can be a specific day number or a range with a hyphen.
  • Month: It can be a specific month number, a range or a comma separated number list.
  • Day (week): It only can be a shorthand for day name
  • Examples:
    • */5 * * * * Run every 5 minutes
    • 5 1 * * Sun Run every Sunday at 1:05 AM
