Test Element Interaction Errors

These elements are designed to provoke "element not interactable: has no size and location" errors

1. Zero-size element (no width/height) - #zero_size_button

2. Element with negative position (off-screen) - #negative_position_button

3. Element with zero opacity (invisible) - #zero_opacity_button

4. Element with display: none - #hidden_button

5. Element with visibility: hidden - #invisible_button

6. Element that appears after delay (initially has no size) - #delayed_button

7. Element inside collapsed container - #collapsed_button

8. Element with transform scale(0) - #scaled_button

9. Element that becomes interactable after 5 seconds - #becomes_interactable_button

10. Element with pointer-events: none - #no_pointer_events_button

11. Element with z-index: -1 (behind other elements) - #behind_element

12. Element that changes size dynamically - #dynamic_size_button
